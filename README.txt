Linux - TP7 : Automatisation de taches avec Bash

## Informations :

@auteurs : Denisa Dudas & Camillia Hammou
@classe : ESGI2

Ce script crée un starter pack pour un projet en langage C.
Que ce soit à l'école ou pour un projet personel, ce script est pratique.

Ce script génére :
- un dossier include
    - un fichier .h
- un dossier sources
    - un fichier .c
- un fichier .gitignore
- un fichier main.c
- un fichier Makefile

Ce script génére egalement le contenu de base de chaque fichier

## Utilisation :

1. Créer un dossier vide
2. Ouvrir le dossier
3. Ouvrir un terminal dans le dossier
4. Taper la commande suivante : "project nomProjet" en remplacant le nomProjet par le nom du projet souhaité
5. Vous avez maintenant un starter pack C prêt à être utiliser
